## 使用方式

1. 閱讀 [評分系統指南](SCORING_GUIDE.md) 了解評分標準
2. 從上方類別中選擇適合的題目
3. 面試時依照每題的追問問題深入了解
4. 使用每題底部的評分紀錄表即時記錄分數

## 評分快速參考

| 分數 | 等級 | 簡述 |
|------|------|------|
| 1 | 不符合標準 | 無具體範例或明顯編造 |
| 2 | 低於期望 | 模糊籠統，缺關鍵 STAR 要素 |
| 3 | 符合期望 | STAR 完整，合格但不突出 |
| 4 | 超越期望 | 具體深入，影響力明確 |
| 5 | 卓越表現 | 極具深度，有量化成果 |

> 詳細說明請見 [評分系統指南](SCORING_GUIDE.md)
